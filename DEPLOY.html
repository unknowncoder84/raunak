1 Build Your App (2 min)

Run these commands in your terminal:

cd frontend npm install npm run build

This creates a build folder ready for deployment.

2 Deploy to Netlify (3 min)

Easiest Method - Drag & Drop:

  1. Go to Netlify Dashboard
  2. Click "Add new site" → "Deploy manually"
  3. Drag the frontend/build folder to the upload area
  4. Wait 1-2 minutes for deployment
  5. Done! You'll get a URL like: https://your-app.netlify.app
